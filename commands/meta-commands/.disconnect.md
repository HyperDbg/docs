# .disconnect \(disconnect from a session\)

### Command

> .disconnect

### Syntax

> .disconnect

### Description

Disconnects from the current session.

### Parameters

None

### Examples

None

### IOCTL

None

### Remarks

1. This function disconnects from a session and will not unload the kernel modules.

None
